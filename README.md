# Python Journal project
A simple Python program that lets users write, save, and view journal entries.
This project is part of my learning journey in Python and Git/GitHub.

## Features
- Add new journal entries
- Save entries to a file
- View previous entries
- Password-protected access
- Light and Dark theme support
- Emoji support in entries
- Simple and begginer-friendly Python code

### Technologies used
- Python 3
- Basic file handling ('open', 'write', 'read')

#### How to run the project
1. Download or clone this repository
2. Open the folder in your code editor
3. Run the python file using: journal.py
